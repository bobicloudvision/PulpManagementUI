# 0.0.1

**Released:** 2026-04-02

Initial release of Pulp Admin UI: Next.js (App Router) admin console for Pulp 3 with React 19, TypeScript, and Tailwind CSS.

### Added

- Dashboard with cached summary counts, refresh, and shortcuts
- Authentication with cookie-based sessions and logout
- Users and groups (list and create)
- RPM and Debian APT repositories: list, create, edit, publish, distribute, delete, content browse, RPM version history and detail
- RPM package content listing and detail; upload flow with chunked uploads and repository association
- Sidebar navigation, administration layout, and error handling in the shell
