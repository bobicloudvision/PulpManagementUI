# 0.0.2

**Released:** 2026-04-02

### Added

- **Workers:** list page and `/api/pulp/workers` proxy
- **Tasks:** paginated task list and `/api/pulp/tasks` proxy
- **Task schedules:** list page and `/api/pulp/task-schedules` proxy
- **Roles (RBAC):** paginated list, create/edit modals, delete for non-locked roles; `/api/pulp/roles` and `/api/pulp/roles/[id]` (GET/POST/PATCH/PUT/DELETE)
- **OpenAPI:** bundled `openapi/pulp.json` for reference
- **Admin footer:** app version from `package.json`, **Changelog** link, and attribution
- **Changelog page:** reads release notes from `changelog/*.md` (semver filenames), renders with `react-markdown`
- **Badge** component used on the users list

### Changed

- **Sidebar:** “Workers & tasks” groups Tasks, Task schedules, and Workers (workers last)
- **Pulp errors:** `pulpFetch` and `readApiDetail` surface Django REST **field** errors and other JSON shapes, not only top-level `detail`
- **PulpRole** TypeScript type aligned with OpenAPI `RoleResponse` (e.g. optional `prn`, `pulp_last_updated`)
- **Debian repository create** request body aligned with Pulp deb/apt API
- **Repository create modal:** naming guidance hidden behind a toggle
